# NovaFeed Frontend

This is the frontend application for NovaFeed, built using React and Vite.

## Tech Stack
- React
- Vite
- Tailwind CSS

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure `.env` with `VITE_API_URL`.
3. Run the development server:
   ```bash
   npm run dev
   ```
